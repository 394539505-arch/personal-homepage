# Personal Homepage

A static personal homepage for Auoung.

## Files

- `index.html`: page structure and content
- `styles.css`: responsive layout and visual design
- `script.js`: small browser interactions
- `assets/hero.png`: homepage hero image

## Local Preview

Open `index.html` directly in a browser.

## Publish

This project is ready for GitHub Pages. Publish the repository and enable Pages from the `main` branch root.
